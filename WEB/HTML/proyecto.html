EL PROYECTO

🥕

¿QUÉ ES?

Collect Carrot es un videojuego 2D de plataformas desarrollado en Godot Engine como proyecto de recerca escolar. El objetivo es recorrer niveles recolectando zanahorias mientras te enfrentas a enemigos.

🎮

MECÁNICAS

El jugador puede moverse a izquierda y derecha, saltar entre plataformas y atacar a los enemigos con la tecla E. Las zanahorias están repartidas por el escenario y hay que recolectarlas todas para avanzar.

🛠️

DESARROLLO

El juego está construido con Godot Engine 4, usando GDScript para la lógica del personaje, enemigos y coleccionables. El arte es pixel art diseñado a medida para el proyecto.

🎯

OBJETIVO

Este proyecto tiene como finalidad aprender los fundamentos del desarrollo de videojuegos: diseño de niveles, programación de físicas, sistema de colisiones y gestión de enemigos con inteligencia artificial básica.

TECNOLOGÍAS UTILIZADAS

GODOT ENGINE 4 GDSCRIPT PIXEL ART TILEMAPS HTML / CSS / JS

FASE 1 — CONCEPTO

Definición de la idea, mecánicas principales y diseño del personaje.

FASE 2 — PROTOTIPO

Primer nivel jugable con movimiento, salto y sistema de coleccionables.

FASE 3 — ENEMIGOS

Implementación de enemigos con patrullas y sistema de combate.

FASE 4 — PULIDO

Arte final, sonidos, niveles adicionales y publicación del proyecto.
